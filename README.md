# KAMACHIX - Academic Analytics Dashboard

KAMACHIX is a web-based dashboard designed to visualize academic and institutional indicators for the University of Nariño.

Originally developed as an academic research project, the platform was later deployed within the university infrastructure and operated in a production environment, providing access to academic analytics and institutional performance metrics through a centralized dashboard.

## Features

* Visualization of academic indicators
* Institutional performance monitoring
* Centralized dashboard for analytics
* Data-driven reporting

## Technologies

* Node.js
* Express.js
* PostgreSQL
* JavaScript
* Jade (Pug)
* HTML
* CSS
* Nodemailer
